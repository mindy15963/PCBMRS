# 꽃가루 농도에 따른 마스크 착용 여부 시스템
3가지 종류(참나무, 소나무, 잡초류)에서 1가지 이상에서 보통 이상이 나오면 마스크 착용 권장이 나오도록 하는 시스템입니다.
사용된 공공데이터는 기상청_꽃가루농도위험지수 조회서비스(3.0)(출처: https://www.data.go.kr/data/15085289/openapi.do)입니다.

# Pollen Concentration-Based Mask Recommendation System (PCBMRS)
This system recommends whether to wear a mask based on pollen concentration levels. If the pollen level is “moderate” or higher in at least one of the three categories—oak, pine, or weeds—the system advises wearing a mask.
The system utilizes public data from the Korea Meteorological Administration’s Pollen Concentration Risk Index Inquiry Service (3.0) (Source: https://www.data.go.kr/data/15085289/openapi.do).
